Data files

Description​

The location of data files is specified within experiment files. Data files typically have three columns containing the following information:

Experiment typeFirst columnSecond columnThird column
CEST / D-CEST / cos-CESTOffset (Hz)IntensityUncertainty
CPMGNumber of cyclesIntensityUncertainty
RelaxationDelay (s)IntensityUncertainty

Data files are peak intensity tables that can be obtained from the output of many peak fitting programs, such as the autoFit subroutine in the NMRPipe program.

  • CPMG
  • Relaxation
